# Macros.fnl (v0.0.1)
Macros for creating lazy sequences.

**Table of contents**

- [`lazy-seq`](#lazy-seq)
- [`lazy-cat`](#lazy-cat)

## `lazy-seq`
Function signature:

```
(lazy-seq ...)
```

Return a lazy sequence, that lazily evaluates the body, and returns a sequence.

## `lazy-cat`
Function signature:

```
(lazy-cat ...)
```

Concatenate arbitrary amount of lazy sequences.
